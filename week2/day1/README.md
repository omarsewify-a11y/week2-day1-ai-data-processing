# Week 2 - Day 1

## Project Overview

This project demonstrates basic data processing using Python.

The program:

1. Loads student support requests from a JSON file.
2. Counts requests by category.
3. Identifies the top three categories.
4. Generates a summary report.
5. Exports the results to a CSV file.

## Files

- data.json
- main.py
- summary.csv

## How to Run

```bash
python main.py
